Quiero una pagina web llamada 
"booksy", la pagina es de libros, tiene que:

- debe estar SOLO en español
- La pagina web debe ser publica PERO no debe dejar hacer prestamo, comentar, califica libro hasta que se registre el usuario 
- tener login y registrar nuevo usuario, esta parte tiene que diferenciarse entre el rol admin y el rol de usuario,
- Los libros se pueden comentar (se debe permitir 1 solo comentario por libro) Esto es opcional para el usuario
- El libro debe tener una opcion (estrellas) para calificarlo y que se haga un promedio. Esto es opcional para el usuario
- tener un apartado donde salgan las categorias de los libros
- tiene que haber un boton donde se haga el prestamo del libro y de ahi que haya una configuracion para poner dias (maximo 90) en caso de que no se configure, entonces que se asigne automaticamente. Al pasar los 90 dias este debe volver a poner el libro en stock es decir, disponible 
- debe tener opcion de ver perfil donde se encuentren los libros que ya haya tenido como prestamo (reportes o historicos) y los que tenga asignados
- tiene que tener un apartado donde aparezcan los destacados, los mas romanticos...
- Si un usuario quiere devolver el libro, tiene que haber un boton para ejecutar esta accion y el libro queda en disponible 
- el usuario administrador debe poder hacer todas las acciones de CRUD y debe ver cuales son los usuarios que estan registrados, activos e inactivos y los prestamos del momento
- Tienen que decir cuántos libros hay disponibles, son máximo 50 libros disponibles por cada libro y a medida que se haga un préstamo este debe descontarse automáticamente (igual que cuando un usuario lo devuelva se debe sumar) en caso de que no haya disponibilidad entonces que aparezca un mensaje
